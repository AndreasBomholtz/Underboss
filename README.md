# Underboss
A bot for the game, Godfather from kabam.com.

## USE AT OWN RISK!

You might get banned by using this bot.

## Usage

To use this script you need:
- Chrome
- Tampermonkey plugin

Then install the Underboss.user.js script.

## Features
- Collect
 - Money
 - Prizes
 - Daileys
- Build
- Research
- Train
- Attack
- Bonds
- Bailout
